# Code of Conduct

このプロジェクトは [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を行動規範として採用します。

日本語訳：[Contributor Covenant 行動規範 v2.1（日本語）](https://www.contributor-covenant.org/ja/version/2/1/code_of_conduct/)

## 連絡先

行動規範違反の報告や相談は、以下までご連絡ください。

- メール：`system@oceans-creative.com`

報告内容は適切に扱い、報告者のプライバシーは保護されます。
