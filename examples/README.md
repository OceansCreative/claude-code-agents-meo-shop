# CLAUDE.md 記入例集（店舗ビジネス向け）

`CLAUDE.md.template` を業種別にどう埋めるかの記入例です。

## 利用方法

自店の業態に近い例を `CLAUDE.md` としてコピーし、自店情報に書き換えてください。

```bash
# 例：飲食店の場合
cp examples/restaurant/CLAUDE.md ./CLAUDE.md
# その後、店名・所在地・売上などを実際の情報に置き換える
```

## 業種別の記入例

| ディレクトリ | 想定業種 | 業態 | カバーする主要観点 |
|---|---|---|---|
| [`restaurant/`](./restaurant/CLAUDE.md) | 飲食店（居酒屋） | 個人事業 | 食品衛生法、原価率、MEO、客単価設計 |
| [`beauty-salon/`](./beauty-salon/CLAUDE.md) | 美容室 | 個人事業 | 美容師法、薬機法、リピート率、Instagram 運用 |
| [`treatment-clinic/`](./treatment-clinic/CLAUDE.md) | 訪問鍼灸院 | 個人事業 | あはき法、広告規制、ケアマネ営業、医療保険 |

## 自店の業種が見当たらない場合

最も近い例をベースに、業態固有のセクションを追加してカスタマイズしてください。例えば：

- ネイルサロン：施術メニュー・所要時間・予約間隔
- 整骨院：施術メニュー・保険適用範囲・自費メニュー比率
- カフェ：客席タイプ別売上・滞在時間・テイクアウト比率
- セレクトショップ：取扱ブランド・客単価・リピート購入率

「この業種の例も追加してほしい」という要望は、Issues か Pull Request で歓迎します。
