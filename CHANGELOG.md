## [1.0.0] - 2026-04-29

### Added

- 6 specialized agents for store-based businesses:
  - `meo-strategist` — Google Business Profile / local search optimization
  - `review-responder` — Review response strategy across platforms (Google, Tabelog, HotPepper, etc.)
  - `local-marketer` — Trade area analysis, flyers, local SEO, community partnerships
  - `sns-operator` — Instagram, LINE Official, X for store businesses
  - `customer-journey` — From discovery to repeat visit, LTV maximization
  - `seasonal-planner` — Annual promotional calendar, seasonal campaigns
- `CLAUDE.md.template` for store-business CLAUDE.md initialization
- Industry CLAUDE.md examples: restaurant (izakaya), beauty salon, treatment clinic

- Bilingual README (Japanese / English)
